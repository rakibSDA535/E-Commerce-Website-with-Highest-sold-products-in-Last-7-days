# E-Commerce-Website-with-Highest-sold-products-in-Last-7-days
ASP. NET Core MVC, SQL Server .Shopping card project in . Net 8 with Repository pattern. Admin and user base Authentication and Authorization. Implement multi-table CRUD operation and user-friendly interface. Include Master-Details and dynamic Dash-Board. Using Advance Store-procedure for find out sold products.
